# Newsletter Writing Skill

## Newsletter Structure

### 1. Subject Line
- Keep under 50 characters for mobile readability
- Create curiosity or promise a specific benefit
- Use numbers when relevant: "5 ways to..." or "3 mistakes that..."
- Avoid spam triggers: ALL CAPS, excessive punctuation, "FREE"
- A/B test when possible — write 3 options, pick the strongest

### 2. Opening Hook (First 2-3 sentences)
- The first line must earn the reader's attention immediately
- Start with a story, question, bold claim, or relatable problem
- Never start with "In this issue..." or "Welcome to issue #47"
- The preview text (first ~100 chars) shows in the inbox — make it count

### 3. Body Content
- **One primary topic per newsletter** — don't dilute focus
- Use short paragraphs (2-3 sentences max)
- Break up text with subheadings, bold key phrases, and bullet points
- Use the inverted pyramid: most important information first
- Include at least one personal anecdote or unique insight
- Write at an 8th-grade reading level

### 4. Call to Action (CTA)
- Every newsletter must have ONE clear primary CTA
- Place it after you've delivered value, not before
- Use action verbs: "Download", "Try", "Read", "Watch"
- Make the CTA visually distinct (button or bold link)
- Secondary CTAs are optional but should not compete with the primary

### 5. Sign-Off
- Brief, warm, and consistent with your brand voice
- Include your name — newsletters are personal
- Optional: P.S. line (one of the most-read parts of any email)

## Tone Rules

- Write as if talking to ONE person, not a crowd
- Use "you" and "your" liberally
- Contractions are encouraged — "you'll", "here's", "don't"
- Be conversational but not sloppy
- Match the energy to the audience: professional for B2B, casual for creators

## Formatting Best Practices

- Paragraphs should be 1-3 sentences
- Use bullet points for lists of 3+ items
- Bold key takeaways so skimmers get value
- Use line breaks generously — dense text loses mobile readers
- Include one image only if it adds meaning (not decoration)

## Engagement Techniques

- Ask a question the reader can respond to
- Share behind-the-scenes or "here's what I learned" moments
- Reference previous issues to build continuity
- Use open loops: tease next week's content
- Social proof: mention reader replies, subscriber milestones

## Common Pitfalls

- ❌ Writing more than 800 words (sweet spot: 300-600)
- ❌ Including more than 3 links (dilutes click-through)
- ❌ Burying the value — readers should get something in the first scroll
- ❌ Inconsistent sending schedule
- ❌ No CTA — always tell the reader what to do next
